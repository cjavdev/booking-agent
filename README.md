# Booking Agent

https://share.descript.com/view/y062As83xh6

## Setup Env

```bash
cp .env.example .env
```

Update with API keys and phone numbers.

## Run with Docker

```bash
docker compose up
```

View the app at http://localhost:3000.

## Development

Start the development server:

`pnpm dev`

Run the tests:

`pnpm test`
